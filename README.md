# 🤖 Voice AI Property Appointment Setter

An AI-powered voice automation solution designed to streamline real estate property visit scheduling through intelligent conversational interactions. The system leverages a VAPI Voice AI Assistant to autonomously handle inbound and outbound calls, engaging prospects in natural, human-like conversations to book appointments seamlessly.

## 🎯 Project Overview

This project is a **Voice AI Appointment Setter** designed for real estate property visit booking.

The system uses:

- VAPI Voice AI Platform  
- Node.js Backend Server  
- Google OAuth Calendar Integration  
- ngrok for Local Development Testing  

The AI assistant can:

✅ Receive customer calls  
✅ Collect name, email, date, and time  
✅ Automatically book property visit appointment  
✅ Create Google Calendar event  
✅ Confirm booking verbally  

---

## 🏗 System Architecture

Architecture Flow :

User Voice Call -> VAPI AI Assistant -> API Request Tool -> Node.js Backend Server -> Google OAuth Authentication -> Google Calendar Event Creation
